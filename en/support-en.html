frequently asked questions

FAQ

Check the power supply (min. 5V/3A for Pi 4, 5V/5A for Pi 5). Is the SD card correctly flashed with Pi Imager and properly inserted? The red LED shows whether power is present.
In Pi Imager: enable SSH before flashing. Or place an empty file named "ssh" (no extension) in the boot partition. Connect with: ssh pi@IP-address
Open terminal and run: sudo apt update && sudo apt full-upgrade -y — then reboot: sudo reboot. Regular updates keep the system secure.
The Pi 5 can reach 80°C under full load. Throttling starts at 85°C. Use the official Active Cooler or a case with heatsink for continuous operation.
At least Class 10 / UHS-I with A1 or A2 rating. Recommended: SanDisk Extreme or Samsung PRO Endurance. For Pi 5: NVMe SSD via M.2 HAT for maximum performance.
Edit /etc/dhcpcd.conf or use nmtui in terminal → "Edit connection" → set IPv4 to "Manual". Restart after saving.
Windows 11 ARM runs on Pi 4/5 via WoA Installer, but with limited performance. Recommended: Raspberry Pi OS, Ubuntu 24.04 or Fedora.
In Pi Imager enter your Wi-Fi credentials before flashing. Or after first boot: sudo raspi-config → System Options → Wireless LAN.
